Styleguide

Iconography

If you need to generate alternate sizes, use the vector versions in the images folder.

Typography

The Playbook uses Whitney HTF as it's typeface. The font files are included in the type folder; see the sytleguide.css for use. Alternate weights may need to be considered for different screen size or resolution. Alternate weights and families are available if needed.

Heading

Sub-Heading

Body text: Used for normal runs of text. Weight, and leading can be adjusted as appropriate for screensize and line length. You'll notice in the printed book the chapter spreads use variations of these guidelines where the body text describing the principles is smaller and heavier for the the white-on-color text, and the header and sub-header are used as reduced size. All of this to say these guidelines are just that, guides, and not rules set in stone.

Alt-body text: Small-cap variant used for supplementary text in side bars or chapter outlines
